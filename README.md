A Python project that solves mazes using **Depth-First Search (DFS)** and **Backtracking** algorithms.
## Features
- Uses **Recursion** to explore paths.
- Implements **Backtracking** to handle dead ends.
## Algorithms Used
- **DFS**: Explores as far as possible along each branch before backtracking.
- **Backtracking**: If a path leads to a wall, the algorithm "backtracks" to the previous intersection.
